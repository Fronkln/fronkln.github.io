Source to Unity

Turkish Page


Source to Unity is a loader based on lewa-j's Unity Source Tools
That can load Source assets to Unity with many big differences:
  • BSP Loader
  • MDL Loader
  • Map entities are loaded
  • Similiar to Source programming(DEFINE_KEYFIELD, Think, Precache etc)
  • Source-like lighting
  • Map input-output
  • Soundscript loader
  • Choreography loader
  • Source phoneme reading from sound

Planned

  • VGUI support?
  • Improved MDL support
  • Mainmenu backgrounds(3D or not)
  • More Source entity support
  • Launch parameter support
